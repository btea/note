### npm配置  
**查看npm配置的远程仓库地址**  
```js
npm config get registry
cnpm config get registry
```
**设置npm远程仓库地址**  
```js
npm config set registry https://registry.npm.taobao.org  
cnpm config set registry https://registry.npmjs.org
```
