子元素的总宽度超出父元素宽度，根据子项收缩宽度公式：
子项收缩宽度 = 子项收缩比例 * 溢出宽度
子项收缩比例 = (子项宽度 x 子项收缩系数(flex-shirnk)) / 所有子项的(宽度 x 收缩系数)之和
结合具体情况：
  • 子项溢出空间的宽度为：500px + 400px - 600px = 300px
  • left收缩宽度：300 * 500 * 2 / (500 * 2 + 400 * 1) ≈ 214.285714285
  • right收缩宽度：300 * 400 * 1 / (500 * 2 + 400 * 1) ≈ 85.714285714
